Задача:
Есть 4 склада, на которые поступают овощи и фрукты по разной цене и в разном кол-ве.
Вывести в файл - st-total.txt:
Список складов может быть выведен в коротком(Название+деньги) или длинном(название+деньги+ абсолютное кол-во товаров) формате с сортировкой по общему в денежном выражении по убыванию:
ST1 - 120000
ST2 - 110000
ST4 - 90000
ST3 - 80000
------------
ST1 - 120000 - 1240
ST2 - 110000 - 2345
ST4 - 90000 - 780
ST3 - 80000 - 2235
Вывести в файл - product-report.txt:
Список продуктов с общей стоимостью на складе, отсортированные по убыванию по имени товара и общей стоимости на складе:
APPLE - ST1 - 65500
APPLE - ST2 - 55500
CABBAGE - ST2 - 3300
CABBAGE - ST1 - 2800
MUSHROOMS - ST2 - 176.80
MUSHROOMS - ST1 - 173.30
MUSHROOMS - ST3 - 172.10

AC:
Все должно разложено по пакетам.
Соответствовать Code convention
Вывод в произвольных форматах сделать через интерфейс
Создать свое исключение и использовать его
В главном методе, передаем исходный файл с данными и формат вывода для информации по складам. 
Например, будет метод, который на вход принимает название файла и цифру:1-короткий формат,2-длинный формат
